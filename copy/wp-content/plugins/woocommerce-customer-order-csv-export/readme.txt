Stable tag: 2.0.2

This plugin adds export functionality to WooCommerce for customers and orders.

== Installation ==

1. Upload the plugin folder to the '/wp-content/plugins/' directory.

2. Activate the plugin  through the 'Plugins' menu in WordPress.

== Usage ==

Go to WooCommerce > Export CSV, select your options and export. Simple!
